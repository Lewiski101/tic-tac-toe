# Tic-Tac-Toe Game

A classic Tic-Tac-Toe game built with HTML, CSS, and JavaScript. Play against another player on the same device.

## How to Play

1.  **Open the Game**: Open the `index.html` file in your web browser.
2.  **Start Playing**: The game board will be displayed. Player X starts the game.
3.  **Take Turns**: Players take turns clicking on an empty square to place their mark (X or O).
4.  **Winning the Game**: The first player to get three of their marks in a row (up, down, across, or diagonally) is the winner.
5.  **Draw**: If all 9 squares are filled and no player has 3 in a row, the game is a draw.
6.  **Restart**: Click the "Restart" button to play again.

## Features

*   Classic 3x3 Tic-Tac-Toe grid
*   Two-player gameplay (X and O)
*   Win, lose, and draw detection
*   Restart functionality
*   Simple and intuitive user interface

## Technologies Used

*   **HTML**: For the structure of the game.
*   **CSS**: For styling the game board and other elements.
*   **JavaScript**: For the game logic and interactivity.
